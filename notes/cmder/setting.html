Cmder使用筆記

安裝

  1. Cmder下載：官網
  2. 解壓縮至想安裝的資料夾，例如我安裝於：D:\Tools\cmder
  3. 新增以上路徑至環境變數 > 系統變數 > PATH

基礎設定

  1. 更改命令提示字元

    2. cmder預設為λ，偶爾會導致有一個字符無法刪除的bug，雖然多餘的字符並不會影響指令運作，但還是建議替換。
    到cmder安裝路徑\vender\clink.lua，將 

    local cmder_prompt = "\x1b[1;32;40m{cwd} {git}{hg}{svn} \n\x1b[1;39;40m{lamb} \x1b[0m"
    改為 
    local cmder_prompt = "\x1b[1;32;40m{cwd} {git}{hg}{svn} \n\x1b[1;39;40m$ \x1b[0m"
    重啟cmder。

Cmder & Git

  1. 設定email與user name

    2. git config --global user.email "misting401@gmail.com"
    git config --global user.name "Astalsi401"
  2. 設定ssh key
    3. ssh-keygen -t rsa -C "misting401@gmail.com"
    ssh key會被儲存於C:\Users\user\.ssh\id_rsa.pub

  3. 將id_rsa.pub中的內容全選並直接複製到Key欄位。
  4. 確認是否成功
    5. ssh -T git@github.com
  • 更改github密碼後，遠端倉庫網址會變更為https，並在每次進行push、pull等指令時要求輸入user.name、password，然而在今年（2021）git已經取消使用密碼進行推送的途徑，在密碼欄位須改為使用Personal access token作為密碼。此時使用以下指令即可改回SSH網址，不需再輸入用戶名與密碼。 
    git remote set-url origin git@github.com:username/repository.git
    確認遠端倉庫網址 
    git remote -v

Cmder & Sublime Text 3

  1. 將Sublime Text 3安裝於cmder安裝路徑\bin\Sublime_Text_3\
  2. 在cmder安裝路徑\config\user_aliases.cmd中加入：
    3. subl="cmder安裝路徑\bin\Sublime_Text_3\sublime_text.exe" $* -new_console:s50H

    subl是用以叫出Sublime的代號，可替換為你習慣的稱呼。-new_console在新分頁開啟。s50占比50%。H橫向排列。縱向排列則為V。

Cmder快捷鍵

Ctrl+T 新頁面
Ctrl+W 關閉頁面
Ctrl+Tab 切換頁面
Alt+Shift+1 開啟cmd.exe
Alt+Shift+2 開啟powershell.exe
Alt+Shift+3 開啟powershell.exe (系統管理員許可權)
Ctrl+n 快速切換到第n個頁籤
Ctrl+R 歷史命令搜尋
Win+Alt+P 開啟設定視窗

其他指令

  • 運行bat：

    cmder /x "/cmd test.bat"

  • scss to css：

    sass --watch main.scss main.css
