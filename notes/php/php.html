容易踩到的坑

  1. php與apache必須為相同位元，如X64。
  2. php8以上，php8apache2_4.dll已經自帶在安裝包中，無須額外下載。
  3. php8以上，httpd.conf載入php的代碼須改為
    4. #載入PHP
LoadModule php_module 'D:/Tools/php/php8apache2_4.dll'

#將PHP配置檔案載入到Apache配置檔案中，共同生效
PHPIniDir 'D:/Tools/php'

#配置Apache分配工作給PHP模組，把PHP程式碼交給PHP處理
#即.php字尾名的檔案
AddType application/x-httpd-php .php
  4. http://localhost/test.php存取被拒時請重新開機。
  5. 連接mysql出現Call to undefined function mysqli_connect()，請參考PHP中出现Call to undefined function mysqli_connect()
