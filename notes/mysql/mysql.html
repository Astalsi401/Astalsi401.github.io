MySQL使用筆記

設定

  1. 安裝後，新增路徑至環境變數 > 系統變數 > PATH
  2. 新增my.ini至安裝目錄，內容為
    3. [client]
# 設定客戶端mysql編碼
default-character-set=utf8
 
[mysqld]
# 設定3306端口
port = 3306
# 設定mysql安裝目錄
basedir=D:\\Tools\\mysql-8.0.27-winx64
# 允許最大連接數
max_connections=20
# 設定服務端編碼
character-set-server=utf8
# 創建新表時使用默認儲存引擎
default-storage-engine=INNODB
  3. 初始化數據庫，執行後將輸出默認密碼
    4. mysqld --initialize --console
  4. 安裝
    5. mysqld install
  5. 啟動
    6. net start mysql
  6. 登錄數據庫
    7. mysql -h 主機名 -u 用戶名 -p
  7. 登錄本機數據庫
    8. mysql -u root -p

常用指令

  1. mysql啟動時使用系統指令需加上system，例如：
    2. system pwd
