vscode筆記

快捷鍵(個人用)

一般

  • run code：ctrl+alt+n
  • stop code run：ctrl+alt+m
  • run sql code：ctrl+enter
  • 隱藏終端機面板：ctrl+js
  • 問題：ctrl+shift+m
  • 輸出：ctrl+shift+'
  • 偵錯主控台：ctrl+shift+y
  • output自動滾動ctrl+alt+r
  • 代碼摺疊：ctrl+hctrl+h
  • 代碼展開：ctrl+jctrl+j
  • markdown、python preview：ctrl+kv
  • 向右分割編輯器ctrl+\
  • 切換分割區域ctrl+123...
  • 變更視窗分割方向：shift+alt+0
  • 多重選取：shift+alt+n
  • 啟用/停用自動換行：alt+z

Quokka

  • start quokka：ctrl+kq
  • stop quokka：ctrl+ke
