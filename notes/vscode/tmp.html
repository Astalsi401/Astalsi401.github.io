vscode筆記

快捷鍵(個人用)

一般

  • 搜尋：Ctrl+F
  • 取代：Ctrl+H
  • 添加註解(行)：Ctrl+/
  • 添加註解(區)：Shift+Alt+A
  • 隱藏終端機面板：Ctrl+JCtrl+S
  • 問題：Ctrl+Shift+M
  • 輸出：Ctrl+Shift+'
  • 偵錯主控台：Ctrl+Shift+Y
  • output自動滾動Ctrl+Alt+R
  • 代碼摺疊：Ctrl+JCtrl+H
  • 代碼展開：Ctrl+JCtrl+J
  • markdown、python preview：Ctrl+KV
  • 向右分割編輯器Ctrl+\
  • 切換分割區域Ctrl+123...
  • 變更視窗分割方向：Shift+Alt+0
  • 多重選取：Shift+Alt+N
  • 啟用/停用自動換行：Alt+Z

Run code

  • run code：Ctrl+Alt+N
  • stop code run：Ctrl+Alt+M
  • run sql code：Ctrl+Enter

Quokka

  • start quokka：Ctrl+KQ
  • stop quokka：Ctrl+KE

Live Sass

  • compile current sass file：Alt+LAlt+S

Live Server

  • change live server workspace：Alt+LAlt+H
  • open live server：Alt+LAlt+Q
  • stop live server：Alt+LAlt+C
