  1. 重量

    2. 影片位置：1:11

    影響操控、加速、剎車，越輕越好，防滾籠通常不需要。

  2. 差速器

    3. 影片位置：2:32

    控制左右輪轉速的同步率，0%代表各自獨立，100%則完全同步。同步率越高，車輛就能傳送越多馬力到輪上，有越快的加速。差速器對轉向過度、轉向不足，出灣加速有相當大的影響。

    1. 前差速器加速（25%）：

      2. 不宜調到太高，雖然使出彎速度較慢，但也會減少過度轉向。

    2. 後差速器加速（40~50%）：

      3. 減少後差速，出彎速度低，但增加抓地；增加後差速，抓地力降低，出彎速度提高。

    3. 差速器減速（0%）：

      4. 屬於個人偏好，不過0%更有利出彎加速。

    4. 中央差速器（70%~90%）：

      5. 決定前後輪分配的馬力，讓車更偏向前驅或後驅。增加後輪馬力，增加轉向過度；但後輪出力過高時將導致加速降低。

    5. 後驅車（45%~80%）

  3. 剎車

    4. 影片位置：4:58

    當啟用abs時，將剎車力度調到100%以上將對剎車速度有負面影響。

  4. 空力

    5. 影片位置：5:40

    減少空力，過彎性能降低，但增加極速。增加空力，犧牲少許極速換取更佳的彎道性能。

    1. 前空力：

      2. 有時候前空力並不適合調到最高（保持在約80%的位置），不然在高速時反而會導致失去抓地力。

    2. 後空力（調到最高）

  5. 阻尼

    1. 回彈硬度（公式決定）

    2. 壓縮硬度（建議為60%）：

      3. 影響避震器面對彈跳時的表現，一般在回彈硬度的50%~70%。

  6. 彈簧

    1. 彈簧（公式決定）

    2. 車身高度：

      3. 降到最低再視情況調高。

  7. 防傾桿（公式決定）

    8. 防傾桿（甩尾）

    1. 前：公式為基礎，視情況調低。
    2. 後：可直接調到最低，再視情況調高。

  8. 輪胎定位

    9. 影片位置：10:50、15:20

    1. 外傾角：

      2. 決定過彎抓地力，調整至踩著油門過彎時外側車輪時接近0°。

      也可以從車輪溫度調整，調整至車輪溫度內側>中間>外側，但差異不要太大（個位數）

    2. 束角（前+0.2 後-0.2）：

      3. 影響轉向能力，但數值與0差距越大，直線時越不穩定。

    3. 前後傾角：

      4. 越高，直線時外傾角越穩定，但過彎時將增加負外傾角。

    輪胎定位（甩尾）

    1. 前外傾角：-5°
    2. 後外傾角：0°
    3. 前束角：5°
    4. 後束角：1.5°
    5. 前後傾角：7°

  9. 齒輪裝置

    10. 影片位置：5:50

    調整至1檔起跑只會碰到紅區1~2次，第3次時就應該上2檔，2檔轉速不宜過高，避免剛換檔就進入紅區。

    齒輪裝置（甩尾）

    1. 方法1：首先找到適合甩位的動力區間，然後把3檔及之後的檔位都調整至接近這個區間。最後齒比看起來會類似下圖：

    2. 方法2：將前四檔調整為Formula Drift系列的齒比，其餘檔位隨意，最後再視駕駛手感調整最終傳動比。

      • 1檔：1.80
      • 2檔：1.45
      • 3檔：1.16
      • 4檔：1.00

  10. 胎壓

    11. 影片位置：18:35

    越高，轉向越靈敏。越低，起步加速越快。最佳操控性會落在33psi左右，調整至暖胎後為33psi。

    觀察輪胎溫度變化，盡量保持正常狀態以維持峰值抓地力（輪胎溫度保持保持在即將變黃之前）。

總結

轉向不足：

  • 調低前防傾桿
  • 調軟前彈簧
  • 增加後輪出力（AWD）
  • 增加前空力
  • 增加前阻尼壓縮硬度
  • 降低前阻尼回彈硬度
  • 降低差速器百分比

轉向過度：

  • 調低後防傾桿
  • 增加後空力
  • 增加後束角
  • 降低後阻尼回彈硬度
  • 增加差速器百分比
  • 將中央差速器調向50%（AWD）

穩定度過低：

  • 條軟前彈簧
  • 減少束角
  • 調整阻尼回彈硬度百分比
  • 調整空力
  • 確認車身高度是否過低

過彎抓地力不足：

  • 確認輪胎溫度、降低胎壓
  • 降低差速器百分比
  • 將中央差速器調向50%（AWD）
  • 調整阻尼回彈硬度百分比
  • 檢查車輪定位
  • 將齒輪比調整為更重視極速（調低齒比）
  • 增加空力
  • 調軟彈簧

公式

(最大值-最小值)×重量比+最小值

以防傾桿為例，賽車防傾桿最大值為65，最小值為1，若車輛前端重量比為45%，則：

前防傾桿：(65-1)*0.45+1=29.8

後防傾桿：(65-1)*(1-0.45)+1=36.2
