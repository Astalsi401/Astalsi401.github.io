007. anova test

單因子變異數分析（One-way Anova）：

  1. 分析的目的是檢驗各組平均值是否相等，而判斷的準則是透過變異數的比較。
  2. 在上例中，在同一班中，樣本的觀測值是不同的，由於這是隨機抽取的案例，因此其中的差異可視為隨機的差異，也稱為隨機誤差。在進行分析時，此類誤差的推算稱之為組內變異數。
  3. 在上例中，各般之間的觀測值也不同。這種差異可能是由於抽樣的隨機性所造成，也可能是由於各班的差異（如老師的教學方法、按學業成績分班的結果）。在此，我們將各班的差異視為系統性的誤差。在進行分析時，此類誤差的推算稱之為組間變異數。
  4. 換句話說，組間變異數（組別效應）愈大，而組內變異數（隨機誤差效應）愈小的話，則F值就愈大，意味著虛無假設被排斥的可能性就愈大。
  5. 當只有兩組時（即J=2），t 分配與Anova的結果是一樣的。即F test比T test更容易拒絕虛無假設，因為t test虛無假設之條件只有兩個，而f test的虛無假設之條件有多個。

di invFtail(組間df, 組內df, CI)

di Ftail(組間df, 組內df, F值)

兩因子變異數分析（Twoway Anova）：

  1. 兩因子變異數分析的重點
    1. ANOVA分析是進入迴歸分析的基礎，不僅可以處理理單一因⼦多組的平均數分析，更可以處理多因⼦下多組平均數的分析。
    2. 本節之分析包括兩個⾃自變項與⼀個應變項。換言之，因子有兩個，而分析的⽬的就是探討這兩個因子是否與應變項有關。此處開始涉及到多因素分析。
    3. 雙因素變異數分析的目的，在於檢驗以下三者：
      • A因子是否與應變項有關（亦稱為主效應 main effects）。即A因素之間各組的平均數不存在差異。
      • B因⼦是否與應變項有關。即B因素之間各組的平均數不存在差異。
      • AB因子間是否存在交互作用效應（interaction effect）。
  2. ⽤圖形顯示兩因子的效應，及其交互作用
    1. A和B因子均無效應：

      2. 顯示無論有沒有使用到A因子，心率⼀直停留在60bps(Beats per minutes)

    2. 只有A因⼦有效應：

      3. 顯示在無論有沒有使用到B因子的情況下，若使用到A因子，則心率會從60bps增加到80bps。

    3. 只有B因⼦有效應：

      4. 顯示在無論有沒有使用到A因子的情況下，若沒有使用到B因子，則⼼率會一直停留在60bps，但若使用了B因子時，則⼼率增加到70bps。

    4. A因子和B因子之間不存在交互作用：

      5. 當沒使⽤到A因子時，若沒使⽤B因子，則心跳停留在60bps，而一旦使⽤到B因子時，則⼼率增加到70bps。換⾔之，在沒使⽤到A因子的狀況下，使用B因子，則心跳會增加10bps。

      當沒使⽤到A因子時，若沒使用B因子，則⼼跳停留在80bps，而一旦使用到B因子時，則⼼率增加到90bps。換⾔之，即是使⽤了A因子，使用B因⼦與否的效應也僅是10bps。和前⾯沒使用A因⼦時一樣。

      因此，A與B之間沒有交互影響。

    5. A因⼦和B因子之間存在交互作用，但主要效應僅在 B 因⼦上：

      6. 這裡心率增加的15bps是A與B交互作用的效應。另外，我們也可以說A因⼦是沒有效應的，但⼀旦增加了B因⼦後，我們不但看到了B因子的效應，也看到了A和B合起來的效應。

    6. A和B都存在著主效應，並且交互作用的效應也存在。
