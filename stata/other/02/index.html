Html & Stata

在本節終將介紹兩種能將Stata執行結果加上文字說明輸出為html檔的方法。

dyndoc

dyndoc的do檔格式如下：


將Markdown語法的文字輸入在這裡，ex:

#標題

##標題2

 - A

 - B

 - C

~~~~
<<dd_do>>

/*將Stata code輸入在這裡, ex:*/
cd "D:/Documents/Stata/Stata/data"
use us_birth_rate, clear
list in 1/8
cd "D:/Documents/Git/stata/other/02"

<</dd_do>>
~~~~

另一段Markdown....

~~~~
<<dd_do>>

/*另一段Stata code*/

<</dd_do>>
~~~~
